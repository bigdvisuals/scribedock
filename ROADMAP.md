# Roadmap

ScribeDock 1.0.0 focuses on a reliable local transcript reader and export workflow for supported YouTube pages.

## Near-Term Polish

- Add public screenshots for the side panel, search, and export flows.
- Keep the Chrome Web Store package checklist current.
- Continue manual QA on videos, Shorts, playlists, and channel tabs.
- Improve documentation when YouTube behavior changes.

## Reliability Improvements

- Expand coverage for playlist and channel edge cases.
- Keep fallback transcript detection resilient when YouTube updates page structures.
- Improve unavailable-transcript messages when YouTube does not expose caption data.

## Future Ideas

- Add more export formatting options.
- Add optional user settings for default transcript language and export format.
- Investigate audio transcription only if it can be done with clear privacy controls and user consent.

## Out Of Scope For 1.0.0

- User accounts
- Payments
- Server-side transcript storage
- AI summaries or audio transcription
