# Changelog

## 1.0.0 - Initial ScribeDock Release

- Renamed and prepared the extension for the ScribeDock 1.0.0 release.
- Added transcript viewing, search, copy, and export workflows for supported YouTube pages.
- Added local package and test commands for release verification.
- Prepared source documentation for GitHub and Chrome Web Store publishing.

## 0.1.0 - Shorts Working Checkpoint

- Added MVP transcript support for YouTube videos with retrievable caption or transcript data.
- Fixed YouTube SPA navigation so the sidebar updates when moving between videos without a full page refresh.
- Added native transcript fallback support for cases where direct caption loading is not available.
- Added YouTube Shorts fallback support for Shorts pages with retrievable transcript data.
